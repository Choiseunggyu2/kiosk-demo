# 🏪 카페 봄날 — 키오스크 교육 데모

어르신 대상 키오스크 사용법 강의를 위한 체험형 데모입니다.

## ✨ 특징
- 순수 HTML/CSS/JS — 설치 없이 브라우저에서 바로 실행
- 반응형: 24인치 세로 키오스크 / 태블릿 / 모바일 자동 감지
- 터치 최적화 (큰 버튼, 큰 글씨)
- 식당/카페 주문 → 결제 → 포인트 적립 전체 흐름 체험

## 🚀 실행 방법
브라우저에서 `kiosk-demo.html` 파일을 열면 바로 실행됩니다.

### Chrome 키오스크 모드 (터치 모니터)
```
chrome --kiosk kiosk-demo.html
```

## 📋 단계 흐름
1. 카테고리 선택
2. 메뉴 선택
3. 장바구니 확인
4. 결제 방법 선택
5. 포인트 적립 (휴대폰 번호, 결제금액 2%)
6. 주문 완료
